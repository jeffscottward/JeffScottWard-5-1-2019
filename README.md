# Jeff Ward - 5/1/2019

## Installation
```bash
npm install
npm run dev
# or
yarn
yarn dev
```

## Run Mocha tests
```bash
npm run test
# or
yarn test
```

## Run Cypress tests
```bash
npx cypress open
# and go to integration folder
# or
npx cypress run --spec "cypress/integration/userInteractions.spec.js"
# or
npx cypress run
# to run everything
```
